# Mingbai Study Room

A standalone Chinese study app split out from Codex Lector.

It focuses on:

- spaced repetition vocabulary review
- pinyin-aware sentence study
- clip-based listening/context notes
- lightweight local-first deck editing

## Local development

```bash
npm install
npm run dev
```

## Current features

- SRS review queue with Again / Hard / Good / Easy
- pinyin modes: off, reveal, always
- example sentences with optional pinyin
- movie/TV clip notes with optional quote pinyin
- local deck editing
- import deck JSON from clipboard
- export deck JSON to clipboard or file
